# TARGET-BRAZIL-E-COMMERCE-ANALYSIS



 Project Overview
 

This project analyzes the **Target Brazil E-commerce dataset**, which contains around 100,000 orders from 2016 to 2018.
The objective is to understand customer behavior, sales trends, logistics performance, and payment patterns using SQL.


This project demonstrates how data analysis can be used to generate actionable business insights and improve decision-making.




 Dataset Description
 
 
 

The dataset includes the following tables:


* Customers
  
* Orders
  
* Order Items
  
* Products
  
* Sellers
  
* Payments
  
* Reviews
  
* Geolocation
  

 Time Period: 2016 – 2018
 
 Total Orders: ~100,000



 Project Objectives
 

* Analyze customer distribution across cities and states
  
* Study order trends (yearly & monthly growth)
  
* Identify seasonality in sales
  
* Evaluate delivery performance and delays
  
* Analyze freight costs across regions
  
* Understand payment methods and installment behavior



 Analysis Performed

Initial Exploration

* Checked data types and structure
  
* Counted total records
  
* Identified unique cities and states
  
* Analyzed data distribution
  

 Order Trends Analysis

* Year-over-year growth in orders
  
* Monthly order trends and seasonality
  
* Time-of-day analysis (Morning, Afternoon, Night)

 Regional Analysis

* Orders by state (month-on-month)
  
* Customer distribution across states
  
* High and low performing regions

Revenue & Cost Analysis


* Total and average order value
  
* Yearly growth in revenue
  
* Freight cost analysis by state
  

 Delivery Performance

* Delivery time analysis
  
* Difference between estimated vs actual delivery
  
* Regions with fastest and slowest delivery

Payment Analysis


* Payment methods distribution
  
* Installment behavior
  
* Monthly trends by payment type



 Key Insights

* Orders increased significantly year-over-year
  
* Strong monthly seasonality observed
  
* Most orders are placed in afternoon and night
  
* Few states contribute to majority of sales
  
* Freight costs vary significantly by region
  
* Some deliveries are faster than estimated
  
* Credit cards are the most preferred payment method
  
* Most customers prefer single installment payments
  

 Recommendations

* Improve logistics in regions with high delivery time
  
* Optimize freight costs in expensive regions
  
* Focus marketing on high-performing states
  
* Run seasonal campaigns during low-demand months
  
* Encourage installment-based payments
  
* Expand seller network in underperforming regions
